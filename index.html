WebDev Assessment This is the assessment. It is both objective and subjective. The goals of this assessment are to get a general understanding of your skill level, attention to detail, and understanding of the web. Requirements Replicate this mockup (yes, the assessment instructions) in HTML/CSS/JavaScript as closely as possible. Use HTML5. Use the "Titillium Web" font. Clicking "Submit!" should create an alert with jQuery The contents of the alert should contain the answer to, "How old is Bobby if, in 4 years, he will be 3 times as old as he is today?" Guidelines Demonstrate knowledge of HTML5 Demonstrate knowledge of efficient CSS Demonstrate knowledge of the DOM Demonstrate knowledge of jQuery Demonstrate knowledge of version control Use performant and secure methods Submit! Using git or git-svn, submit a Github pull request to the original repository for our review. If you are not able to submit a pull request, email us your zipped project.
